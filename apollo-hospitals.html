Apollo Hospitals – Demand Forecasting

Problem

Apollo Hospitals wanted to forecast SKU-level sales and optimize inventory to reduce stockouts and overstock losses.

Approach

  • Analyzed 2 years of sales & seasonal data using SQL & Excel.
  • Built forecasting models using Python (ARIMA, Prophet).
  • Created Power BI dashboards to track stockout rates, sales trends, and turnover.
  • Integrated Oracle DB with Power BI Gateway for automated refreshes.

Tools Used

Python (Prophet, ARIMA), SQL, Power BI, Excel, Oracle

Business Impact

  • Reduced stockouts by 28%.
  • Cut overstock losses by ₹1.2 Cr/year.
  • Enabled proactive restocking decisions during promotions.
